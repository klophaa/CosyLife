# CosyLife

Application web mobile installable sur Android pour simplifier la vie au quotidien.

## Premiere version

- To do list modifiable avec progression du jour.
- Gestion simple du budget avec entrees, depenses et reste.
- Listes personnalisables pour courses, valise, rendez-vous ou rappels.
- Notes rapides.
- Ambiance cosmique avec variante nature.
- Sauvegarde locale dans le navigateur.
- PWA installable, orientation verticale, compatible Vercel.

## Lancer en local

```bash
npm run dev
```

Puis ouvrir `http://127.0.0.1:4173`.

Si PowerShell bloque `npm`, lancer directement:

```bash
node server.js
```

## Deploiement Vercel

1. Pousser ce dossier sur GitHub.
2. Importer le repository dans Vercel.
3. Garder les reglages par defaut pour un site statique.
